## Training flow (each sub-pattern)

1. **Prompt X-Ray** — highlight signals/constraints in real LC wording  
2. **Constraint budget** — chips this leaf requires  
3. **Say it out loud** — 15s interview script + practice mode  
4. **Template slots** — Mad Libs base code + per-LC fills  
5. **Variation helix** — evolution chain (167 → 15 → 18…)  
6. **Mistake autopsy** — wrong code, killing test, one-line fix  
7. **Topic receipt** — mark complete, export summary  

Plus: **family card grid**, **global constraint explorer**, **must-know table**.
